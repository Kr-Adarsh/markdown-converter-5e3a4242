Markdown Viewer Application

This single-file HTML application dynamically renders Markdown content using marked.js and provides syntax highlighting for code blocks using highlight.js.

Key Features:
  • Tabbed Interface: Easily switch between the fully rendered HTML output and the original Markdown source.
  • Dependencies: Utilizes Bootstrap 5 for responsive layout and tab functionality (loaded via CDN).
  • Self-Contained: All functionality works when opened directly in a browser.
